Dhruv Chandna and Luke Flecker's Senior Research Project

Description:
This project uses AI and machine learning to create a more transparent model for calculating insurance premiums, aiming to reduce customer complaints. The model improves the interpretability of pricing calculations, ensuring fairness, accuracy, and building trust in the insurance process.

Setup:
Once you've downloaded the repository to your local machine, open the folder in your source code editor. After that, install the required libraries needed to run the files.

Usage:
Navigate into the src folder, then the gui folder, and run the interactive_gui.py file on your local network. Click the link in the terminal to open up our locally hosted user interface on your default browser.
