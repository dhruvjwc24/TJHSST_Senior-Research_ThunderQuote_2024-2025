How the research explorer works

Synthetic research simulator, not an insurance quote.

This site explains a student research artifact. It does not estimate loss probability, property safety, financial value, or an amount anyone should pay.

Purpose

ThunderQuote Research Explorer makes a legacy senior-research dataset inspectable. It preserves the original geography and generated-event workflow while replacing an unsupported dollar result with a dimensionless scenario index. The goal is transparency: a reviewer can see the inputs, formula version, data period, and exact artifact identity.

Illustrative Scenario Index

For each county, generated claims and generated storms are summed over 1980–2023 and divided by 44. The neutral county score is 60% of its claims rate relative to the national county median plus 40% of its storm rate relative to that median. The score is multiplied by the visible scenario factors and shown relative to 100. It is not clamped into a severity band. 

base = 0.6 × claims_rate / median_claims_rate
     + 0.4 × storms_rate / median_storms_rate

index = round_half_away_from_zero(100 × base × scenario_multiplier, 1)

A value of 100 means the generated county inputs and selected factor product align with the formula's reference. It does not mean average real-world risk. Every coefficient is published in the deployed formula.json file and copied into the research record export.

Geography coverage

The frozen source covers 50 states. The build joins names to Census GEOIDs, resolves seven reviewed historical/punctuation aliases, and withholds six source keys that do not distinguish a county from an independent city: Baltimore, St. Louis, Fairfax, Franklin, Richmond, and Roanoke. Those geometries remain visible as unavailable. DC and territories are outside the source coverage.

Limitations and prohibited uses

  • The event values were generated with an unseeded random process and are not observations.
  • The scenario coefficients were not trained, calibrated, or validated against outcomes.
  • No exposure denominator, uncertainty interval, actuarial validation, or fairness evaluation exists.
  • Do not use the site for insurance, underwriting, lending, purchasing, pricing, or safety decisions.
  • The repository does not yet state redistribution rights or a software license; public production promotion requires owner review.

Reproduce a result

  1. Use the explorer's “Download JSON” action after selecting a county and scenario.
  2. Match its dataset ID and formula version to the values below.
  3. Run npm ci and npm run check from an existing checkout.
  4. Maintainers can rebuild frozen artifacts with the separately pinned Python environment and npm run artifacts:build.
