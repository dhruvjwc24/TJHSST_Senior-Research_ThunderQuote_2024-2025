# Security policy

This is an archived research project with a static browser application. Do not include secrets, personal data, proprietary property records, or production credentials in issues, commits, scenario exports, or deployment configuration.

Report a vulnerability privately through GitHub's repository security-advisory feature when available. If it is unavailable, contact the repository owner through their GitHub profile without publishing exploit details first.

Supported surface: the latest commit on the default branch and its exact Vercel preview. Historical Python research scripts are retained for provenance and are not production services.
